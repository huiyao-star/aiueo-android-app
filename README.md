# AIUEO Android App 🍙

![Android Version](https://img.shields.io/badge/Android-8.0%2B-brightgreen.svg)
![Kotlin](https://img.shields.io/badge/Kotlin-1.9%2B-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg) ## 📖 Overview

The official Android application for the **AIUEO Japanese Learning Platform** ([aiueo.cc](https://aiueo.cc)). This app brings the comprehensive Japanese learning experience to mobile, allowing users to practice pronunciation, vocabulary, grammar, and daily conversations on the go.

## ✨ Features

* **五十音 (Gojūon):** Interactive Hiragana and Katakana charts with stroke order and audio pronunciation.
* **单词 (Vocabulary):** Flashcards and spaced repetition for effective vocabulary memorization.
* **语法 & 基础 (Grammar & Basics):** Structured lessons covering core Japanese grammar points.
* **对话 (Conversations):** Real-life dialogue practice scenarios.
* **教学视频 (Video Tutorials):** Integrated video player for comprehensive learning sessions.
* **Multi-language Support:** Native support for Simplified Chinese interface.

## 🛠 Tech Stack

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (or XML layouts if using the View system)
* **Architecture:** MVVM (Model-View-ViewModel) / Clean Architecture
* **Networking:** Retrofit2 + OkHttp
* **Asynchronous Programming:** Kotlin Coroutines & Flow
* **Local Storage:** Room Database / DataStore
* **Dependency Injection:** Hilt / Dagger

## 🚀 Getting Started

Follow these steps to build and run the project on your local machine.

### Prerequisites

* [Android Studio](https://developer.android.com/studio) (Latest stable version recommended)
* JDK 17+
* Android SDK (Minimum API level 24 / Target API level 34)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/huiyao-star/aiueo-android.git](https://github.com/huiyao-star/aiueo-android.git)
    ```
2.  **Open the project:**
    * Launch Android Studio.
    * Select **File > Open** and navigate to the cloned `aiueo-android` directory.
3.  **Sync Gradle:**
    * Wait for Android Studio to download dependencies and sync the project.
4.  **Run the App:**
    * Connect an Android device via USB (with USB Debugging enabled) or start an Android Emulator.
    * Click the **Run** ▶️ button in the top toolbar.
